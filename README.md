## How it was made
Built with a deep-research agentic flow: 22 sources fetched, 96 claims extracted, 25 adversarially
verified (19 confirmed / 6 refuted), synthesized 7 June 2026.

Confidence labels (High / Medium / Unverified) reflect source consensus at research time.
The weapon meta is time-sensitive — re-verify before relying on specific guns.
